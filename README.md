# Custom-mod-painting-1.21.5
Adds the /CustomPainting help command, letting players replace any painting or texture with custom images or GIFs in Minecraft. Easily personalize your world with unique visuals and animated artwork.

# 🎨 CustomPainting Mod

Bring your Minecraft world to life with custom images and GIFs!

## ✨ Features
- Replace any in-game painting with your own images  
- Support for animated GIFs  
- Simple command system  
- Fully customizable visuals  

## 🚀 Getting Started
1. Install the mod on **Minecraft 1.21.5+ (Forge)**  
2. Launch the game  
3. Use the command:/CustomPainting help
4. Follow the instructions to add your custom images or GIFs  

## 📦 Installation
- Download the mod `.jar` file  
- Place it in your `mods` folder  
- Make sure you are using **Forge 1.21.5+**  

## 🎯 Usage
Use the built-in help command to learn how to:
- Replace paintings  
- Load custom images  
- Apply GIF animations  

## 🛠 Requirements
- Minecraft **1.21.5+**  
- Forge Mod Loader  

## 💡 Notes
- Works with most resource packs  
- Large images or GIFs may affect performance
